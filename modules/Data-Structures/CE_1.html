Vectors

  1. Create four vectors each with at least five values of your choosing. Each vector must have a different class type and should be assigned to an object with an (at least semi-) informative name. Show the code that created the vector, showed the contents of the vector, and demonstrated the class of the vector.

  2. With one of your vectors from the first question, demonstrate code that will show one specific item in the vector.

  3. With one of your vectors from the first question, demonstrate code that will show multiple specific items in the vector.

 

Data Frames I

  1. Create a data frame that has at least three related variables recorded on at least five individuals. You may use the same variables/data as in the “Vectors” question if the data in your vectors are related. Show the code that created the data frame, showed the full contents of the data frame, and showed the structure of the data frame.

  2. Show how to find one specific value in your data frame.

  3. Show how to isolate a single row from your data frame.

  4. Show how to isolate multiple rows from your data frame.

  5. Show three methods to isolate a single variable from your data frame.
